# slack bot template

## 使用技術

- Nodejs
- TypeScript
- Firebase Functions
- (express)
- (@slack/bolt)

## 動作

このbotは

- 「/test」コマンドで"test"を返す
- "ahiahi"というメッセージに反応して"Hello"を返す

を行うことが可能です
